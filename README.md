# 📊 Superstore Sales Dashboard (Power BI)

This project analyzes a retail dataset (9,000+ records) to uncover business insights using Power BI.

## 🔍 Key Insights
- Total Sales: 2.3M
- Total Profit: 286K
- Technology category is most profitable
- Furniture has high sales but low profit

## 📈 Features
- KPI cards (Sales, Profit, Quantity)
- Category-wise performance
- Region-wise distribution
- Top products analysis
- Interactive slicer (Region)

## 🛠 Tools Used
- Power BI
- Python (Pandas)
